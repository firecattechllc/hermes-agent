# Sigil Step 1 — Hermes Application Foundation

## Decision

Sigil is an application on Hermes, not a second orchestration platform.

## Ports

The application depends on four initial Hermes capabilities:

1. Graph context
2. Long-term memory recall
3. Governed model analysis
4. Evidence recording

Transport and runtime details remain outside Sigil's domain layer.

## Governance

This milestone performs no live trades, brokerage mutations, spending, publishing,
deployments, or unrestricted shell execution. Analysis is advisory and evidence-backed.
Any future execution capability requires explicit Hermes approval policy and user authorization.

## Next milestone

Connect the ports to real Hermes services and add the Financial Intelligence Foundation:
company identity, market-data provider abstraction, FinBERT sentiment, provenance, freshness,
and the first production-grade company research report.
